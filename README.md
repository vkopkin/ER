Для решения задачи был сгенерирован случайный граф Эрдёша–Реньи с параметрами n=40 и p=0,45 с использованием библиотеки NetworkX. Средняя степень вершины была вычислена как отношение суммы степеней всех вершин к их количеству. Полученный результат сравнивался с теоретическим значением средней степени вершины для модели Эрдёша–Реньи, которое определяется по формуле (n−1)p. Сравнение показало, что экспериментальное значение близко к теоретическому, а небольшие отклонения обусловлены случайным характером генерации графа.
